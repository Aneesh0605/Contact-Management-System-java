# Contact Management System (Core Java)

A menu-driven Contact Management System built using Core Java and the Collections Framework.

## Features
- Add new contacts
- View all contacts
- Search contacts by name
- Search contacts by phone number
- Update contact details
- Delete contacts
- Uses TreeMap for sorted contact storage
- In-memory storage (data clears after program exit)

## Technologies Used
- Java
- OOP Concepts
- Collections Framework (TreeMap)

## How to Run
1. Compile the program:
   javac Main.java
2. Run:
   java Main
